# WebGL Context

Activates WebGL/3D development mode. See full reference: `profiles/webgl.md`

## Quick Reference
- R3F Canvas setup with `dpr`, `gl` options
- GSAP dynamic import (no SSR) + `useGSAP` hook
- Lenis + ScrollTrigger integration
- Performance budgets: 60fps target, <10ms JS per frame
- Always dispose geometries/materials on unmount

For detailed patterns and examples: `/docs three`, `/docs gsap`, `/docs lenis`
