# React Native Context

Activates React Native/Expo development mode. See full reference: `profiles/react-native.md`

## Quick Reference

- **Expo Router** for file-based navigation
- **FlashList** for performant lists (not FlatList)
- **Reanimated** for 60fps animations
- **Safe area** handling on all screens
- **Platform-specific** code for iOS/Android differences

## Priorities

1. Stability - no crashes
2. Performance - 60fps target
3. Platform feel - iOS feels iOS, Android feels Android

For detailed patterns, check the full profile.
