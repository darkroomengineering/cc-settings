## Methodology

Per `skills/consolidate/SKILL.md`:

1. **Trigger overlap audit** ‚Äî read every skill description, group by intent, flag duplicates.
2. **Rule contradiction audit** ‚Äî read every rule, flag opposite advice and topic overlap.
3. **Discoverability audit** ‚Äî verify every skill has either a "Triggers" list or an explicit slash invocation.
